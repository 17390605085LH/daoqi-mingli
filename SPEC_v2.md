# 命理服务网站 · 设计规范 (SPEC.md) v2.0

---

## 变更说明 (v2.0)

基于用户反馈，做以下调整：

### 1. 服务方案重新设计

**旧方案问题：**
- 只有3档，全部是"付费门槛较高"的一次性服务
- 缺乏入门级/免费体验，年轻人难以尝试
- 没有体现"真人专家"差异化价值

**新方案设计理念：AI + 真人双轨制**

核心逻辑：
- **AI层**：标准化分析，降低成本，覆盖大众
- **真人层**：高溢价服务，解决"AI能做的我们都能做，但有些事只有真人能做到"

---

### 3. 差异化价值定位（核心）

**口号升级：道器合一 · 数智命理 + 真人大师**

**为什么选择我们，而不是自己去用AI？**

| AI能做的 | 真人才能做的 |
|---------|-------------|
| 排盘、八字解析 | 复杂命局的综合判断 |
| 五行统计 | 解读"命"背后的"运"与"势" |
| 标准趋避建议 | 因人而异的个性化方案 |
| 固定模板报告 | 当场互动、追问、深度挖掘 |
| - | 调理方案：药方/风水/养生/改名 |
| - | 情感/事业关键抉择的真人判断 |
| - | 山医堂主：身体+命理双重诊断 |

**山医堂主** 是我们的核心差异化——不是纯算命，而是"医命双修"，能调理身体、化解运势，这是纯AI完全无法替代的。

### 4. 内容解锁逻辑

```
免费尝鲜
    ↓ 转化
学生党/上班族（¥19-49）
    ↓ 复购/升级
进阶/专业版（¥99-199）
    ↓ 高净值用户
真人专家层（¥399-2999）
```

报告页底部升级CTA逻辑：
1. 先展示免费尝鲜内容（八字+简运）
2. 按用户选择解锁更多AI模块
3. 真人服务作为高阶选项单独呈现
